# 🚂 HAT YAI Journey

A collaborative web application project developed to streamline data management and deliver a seamless user experience. This project demonstrates a complete team-based development cycle, integrating a robust headless CMS backend with a responsive frontend interface.

## 👤 My Role & Contributions: Backend Development
As the dedicated **Backend Developer** for this group project, my primary focus was on architecting the data layer, managing content structures, and providing reliable APIs for the frontend team. My specific contributions included:
*   **Headless CMS Architecture:** Configured and deployed a **Strapi** backend environment (running on Node.js) to serve as the central content management system.
*   **API & Data Management:** Designed database schemas using Strapi's Content-Type Builder, which automatically generated RESTful API endpoints for the frontend developers to consume.
*   **Security & Authentication:** Managed role-based access control (RBAC) and user authentication workflows using Strapi's built-in security plugins to protect sensitive data.
*   **Server Maintenance:** Handled the configuration and deployment of the backend server to ensure high availability and seamless data synchronization.

## 🛠️ Tech Stack & Tools
*   **Backend (My Focus):** Node.js, Strapi (Headless CMS), RESTful APIs
*   **Database:** SQLite / PostgreSQL (managed via Strapi)
*   **Frontend (Team Focus):** HTML, CSS, JavaScript / Web Framework
*   **Tools:** Git, Postman (for API testing)
